Privacy Policy

Wayback Recon Tool is a Chrome extension designed to help users passively retrieve historical URLs and query parameters from the Internet Archive's Wayback Machine.

Data Collection

This extension does not collect, store, or transmit any user data. We do not collect:

  • Personally identifiable information (e.g., name, email, address)
  • Passwords or authentication data
  • Browsing history, user activity, or interaction behavior
  • Location or device-specific information
  • Website content, keystrokes, or mouse events

Purpose

The extension only accesses the current tab's domain to query publicly available archived data from web.archive.org.

Data Sharing

We do not share, sell, or transfer any user data to third parties. All functionality is local to the user's browser.

Third-party Services

This extension only makes requests to the Wayback Machine API (https://web.archive.org) to fetch publicly archived URLs.
